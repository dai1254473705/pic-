BrickWall
===


类似360，百度等图片搜索结果页面的效果的布局插件；正在开发中

##如何使用

```javascript
    // #demo最好定宽
    $("#demo").brickwall();
```

##API

- brickMargin;
    
    图片外边距；
    ```javascript
    $("#demo").brickwall({brickMargin: 10});
    ```

- baseRowHeight;

    基准高度（在此基础上做调整）；
    ```javascript
    $("#demo").brickwall({baseRowHeight: 150});
    ```
